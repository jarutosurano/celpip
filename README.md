# CELPIP Study Guide

A comprehensive CELPIP exam preparation study guide built with [MkDocs Material](https://squidfunk.github.io/mkdocs-material/).

## Live Site

**[jarutosurano.github.io/celpip](https://jarutosurano.github.io/celpip/)**

## Sections

- **Getting Started** — Introduction, test overview, and study plan
- **Listening** — Strategies and practice tips for all 6 parts
- **Reading** — Speed reading techniques and comprehension strategies
- **Writing** — Templates for Task 1 (Email) and Task 2 (Survey)
- **Speaking** — Tips for all 8 speaking tasks
- **Resources** — Free and paid resources, YouTube channels, language boosters

## Development

```bash
# Install dependencies
pip install -r requirements.txt

# Start dev server
mkdocs serve

# Build for production
mkdocs build --strict
```

## Deployment

Automatically deployed to GitHub Pages on push to `main` via GitHub Actions.
